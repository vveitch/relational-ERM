The Homo-Sapiens dataset is from BioGRID (https://downloads.thebiogrid.org/BioGRID), with some processing applied.

For more information see: Stark C, Breitkreutz BJ, Reguly T, Boucher L, Breitkreutz A, Tyers M. Biogrid: A General Repository for Interaction Datasets. Nucleic Acids Res. Jan 1, 2006; 34:D535-9
